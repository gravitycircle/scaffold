== Description ==

Creates a custom field for Dashicons. Requires ACF.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-kld-dashicon` folder into your `wp-content/plugins` folder
2. Activate the Dashicon plugin via the plugins admin page
3. Create a new field via ACF and select the Dashicon type
4. Read the description above for usage instructions
