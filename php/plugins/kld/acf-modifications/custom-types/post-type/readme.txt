== Description ==

Creates a custom field for post types. Requires ACF.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-kld-post-type` folder into your `wp-content/plugins` folder
2. Activate the Post Type plugin via the plugins admin page
3. Create a new field via ACF and select the Post Type type
4. Read the description above for usage instructions
